PHP Product Inventory Management System

INSTALLATION:
1. Extract all files to your web server directory (e.g., htdocs, www, public_html)
2. Ensure PHP is installed and running
3. Make sure the directory has write permissions for creating products.json
4. Open index.php in your web browser

FEATURES:
- Add products with name, quantity, and price
- Automatic calculation of total value (quantity × price)
- Display all products ordered by submission datetime
- Grand total calculation at the bottom
- Edit functionality for each product
- Delete functionality for each product
- Ajax-based form submission (no page reload)
- Bootstrap responsive design
- Data stored in JSON format

REQUIREMENTS:
- PHP 7.0 or higher
- Web server (Apache, Nginx, etc.)
- Modern web browser with JavaScript enabled

FILES INCLUDED:
- index.php: Main page with form and product display
- api.php: Backend API for CRUD operations
- script.js: JavaScript for Ajax and UI interactions
- README.txt: This file
